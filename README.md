# A small rendering engine built from scratch to understand how modern graphics pipelines work.

Features:
- glTF model loading
- VAO/VBO/EBO mesh rendering
- Perspective camera
- Model/view/projection transformations
- Texture mapping
- Phong lighting
- Point / spotlight / direct lighting
